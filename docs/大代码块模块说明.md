## 🔧 技术实现

### 模块结构
```
large-code-blocks.js
├── LANGUAGE_LABELS (常量)
├── ensureCodeBlockWrapper()
├── detectCodeLanguage()
├── extractLanguageToken()
├── toLanguageLabel()
├── showCopySuccess()
├── fallbackCopy()
├── injectLargeCodePatchStyles()
└── initLargeCodeBlocks() (导出)
```

### 加载顺序
1. 模块文件先加载（无 defer）
2. 其他脚本延迟加载（defer）
3. DOMContentLoaded 时初始化

### 错误处理
- 所有初始化都包含 try-catch
- 失败时输出警告但不阻断页面
- 提供降级方案（旧版复制方法）
