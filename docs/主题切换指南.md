# 博客主题切换使用指南

## 概述

博客系统现在支持三种主题风格，可以在后台管理界面即时切换，无需重启服务。

## 可用主题

### 1. Default 风格（默认）
- 原有的默认风格
- 深色渐变背景
- 侧边栏固定
- 适合喜欢原始设计的用户

### 2. Indigo 风格（推荐）⭐
- 参照 [yscoder.github.io](https://yscoder.github.io/) 的优雅设计
- 左侧固定侧边栏（深色背景 #34495e）
- 右侧白色卡片式内容区域
- Material Design 风格
- 简洁优雅的排版
- 响应式设计，移动端侧边栏可折叠

**特点：**
- 左侧侧边栏包含站点信息、导航菜单、社交链接
- 右侧内容区域最大宽度 900px 居中显示
- 卡片式文章列表，悬停有阴影效果
- 顶部渐变色 Banner 展示页面标题
- 移动端友好，侧边栏可通过按钮切换

### 3. Hexo 风格
- 简洁清新的卡片式布局
- 类似 Hexo 博客系统
- 顶部固定导航栏
- 侧边栏展示作者信息、标签云、归档列表
- 天蓝色主色调

### 4. Astro 风格
- 现代简约设计
- 清爽的视觉风格
- 适合技术博客

### 5. Vercel 风格
- Vercel 官网风格
- 极简设计
- 黑白配色为主

## 如何切换主题

### 方法一：后台管理界面（推荐）

1. 登录管理员账户
2. 进入"后台管理"
3. 在"基本设置"部分找到"博客主题风格"下拉菜单
4. 选择你想要的主题：
   - 默认风格
   - Indigo 风格（推荐）
   - Hexo 风格
   - Astro 风格
   - Vercel 风格
5. 点击"保存设置"按钮
6. 刷新博客页面即可看到新主题

**优点：**
- 即时生效，无需重启服务
- 可以随时切换
- 操作简单直观

### 方法二：配置文件（不推荐）

编辑 `config.yaml` 文件：

```yaml
# 博客主题
blog_theme: "indigo"  # 可选值: default, indigo, hexo, astro, vercel
```

**注意：** 此方法需要重启服务才能生效，不如后台管理方便。

## 主题自定义

### Indigo 主题自定义

如果你想自定义 Indigo 主题的颜色，编辑 `static/css/blog-indigo-new.css` 文件：

```css
:root {
    --sidebar-bg: #34495e;      /* 侧边栏背景色 */
    --link-color: #3f51b5;      /* 链接颜色 */
    --content-bg: #f8f9fa;      /* 内容背景色 */
    --card-bg: #ffffff;         /* 卡片背景色 */
    --text-primary: #34495e;    /* 主要文字颜色 */
    /* ... 更多变量 */
}
```

### Hexo 主题自定义

编辑 `static/css/blog-hexo.css` 文件：

```css
:root {
    --primary: #49b1f5;      /* 主色调 */
    --accent: #ff7242;       /* 强调色 */
    --bg: #f5f7fa;          /* 背景色 */
    --text: #2c3e50;        /* 文字色 */
    /* ... 更多变量 */
}
```

## 技术实现

主题切换功能的实现原理：

1. **数据库存储**：主题设置存储在 `system_settings` 表中
2. **动态模板选择**：根据主题设置动态选择对应的模板文件
3. **即时生效**：每次请求都从数据库读取当前主题设置
4. **无需重启**：不依赖配置文件，修改后立即生效

## 常见问题

### Q: 切换主题后样式没有变化？
A: 请清除浏览器缓存（Ctrl+F5 或 Cmd+Shift+R）后刷新页面。

### Q: 可以同时使用多个主题吗？
A: 不可以，同一时间只能使用一个主题。但可以随时切换。

### Q: 自定义 CSS 后如何恢复默认？
A: 从 Git 仓库重新拉取对应的 CSS 文件即可。

### Q: 移动端显示不正常？
A: 所有主题都支持响应式设计，如果显示异常请清除缓存或检查浏览器兼容性。

### Q: 主题切换会影响文章内容吗？
A: 不会，主题只影响页面显示样式，不会影响文章内容和数据库数据。

## 推荐配置

根据不同使用场景，推荐以下主题配置：

- **个人博客**：Indigo 风格（优雅专业）
- **技术博客**：Astro 风格（现代简约）或 Hexo 风格（简洁清新）
- **企业博客**：Vercel 风格（极简专业）
- **文档站点**：Default 风格（传统稳重）
