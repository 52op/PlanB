# 代码块复制功能说明

## 功能概述

为三种博客主题（Default、Indigo、Hexo）的文章详情页添加了代码块一键复制功能。

## 功能特性

### 1. 自动检测
- 页面加载时自动扫描所有 `<pre>` 标签
- 为每个代码块添加复制按钮
- 不会重复添加（已有按钮的代码块会被跳过）

### 2. 智能显示
- **PC 端**：鼠标悬停在代码块上时显示复制按钮
- **移动端**：复制按钮始终显示，方便触摸操作
- 按钮位置：代码块右上角

### 3. 复制功能
- 点击按钮复制整个代码块内容
- 支持现代浏览器的 Clipboard API
- 自动降级到 `document.execCommand('copy')` 兼容旧浏览器
- 复制成功后显示视觉反馈

### 4. 视觉反馈
- **默认状态**：显示复制图标 + "复制"文字
- **复制成功**：图标变为对勾 + "已复制"文字，按钮变绿色
- **2秒后**：自动恢复到默认状态

## 技术实现

### HTML 结构

```html
<!-- 原始代码块 -->
<pre><code>console.log('Hello World');</code></pre>

<!-- 添加按钮后 -->
<div class="code-block-wrapper">
    <button class="code-copy-btn">
        <svg class="copy-icon">...</svg>
        <svg class="check-icon" style="display: none;">...</svg>
        <span class="copy-text">复制</span>
    </button>
    <pre><code>console.log('Hello World');</code></pre>
</div>
```

### JavaScript 逻辑

```javascript
// 1. 查找所有代码块
const codeBlocks = document.querySelectorAll('pre');

// 2. 为每个代码块创建包装器和按钮
codeBlocks.forEach(function(pre) {
    const wrapper = document.createElement('div');
    wrapper.className = 'code-block-wrapper';
    
    const copyBtn = document.createElement('button');
    copyBtn.className = 'code-copy-btn';
    // ... 添加图标和文字
    
    // 3. 添加点击事件
    copyBtn.addEventListener('click', function() {
        const text = pre.textContent;
        navigator.clipboard.writeText(text);
        showCopySuccess(copyBtn);
    });
});
```

### CSS 样式要点

```css
/* 包装器使用相对定位 */
.code-block-wrapper {
    position: relative;
}

/* 按钮使用绝对定位 */
.code-copy-btn {
    position: absolute;
    top: 8px;
    right: 8px;
    opacity: 0; /* 默认隐藏 */
}

/* 悬停时显示按钮 */
.code-block-wrapper:hover .code-copy-btn {
    opacity: 1;
}

/* 移动端始终显示 */
@media (max-width: 768px) {
    .code-copy-btn {
        opacity: 1;
    }
}
```

## 浏览器兼容性

### Clipboard API（推荐）
- Chrome 63+
- Firefox 53+
- Safari 13.1+
- Edge 79+

### execCommand（降级方案）
- 支持所有现代浏览器
- IE 9+

## 使用示例

### Markdown 代码块

```markdown
\`\`\`javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
greet('World');
\`\`\`
```

渲染后会自动添加复制按钮。

### HTML 代码块

```html
<pre><code class="language-python">
def hello():
    print("Hello, World!")
</code></pre>
```

也会自动添加复制按钮。

## 样式定制

### Default 主题
- 灰色背景（`var(--bg-gray)`）
- 悬停时变为主题色（`var(--link-color)`）
- 简洁的边框和圆角

### Indigo 主题
- 半透明白色背景
- 悬停时变为主题色，带向上位移动画
- 阴影效果更明显

### Hexo 主题
- 白色背景
- 悬停时变为主题色
- 圆角设计（`var(--radius)`）

## 注意事项

1. **代码块嵌套**：只会为最外层的 `<pre>` 标签添加按钮
2. **动态内容**：如果页面动态加载代码块，需要重新调用 `initCodeCopyButtons()`
3. **样式冲突**：确保没有其他 CSS 覆盖 `.code-copy-btn` 的样式
4. **移动端**：按钮文字在移动端隐藏，只显示图标
