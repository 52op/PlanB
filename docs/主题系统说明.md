# 博客主题系统

## 快速开始

### 1. 切换主题（推荐方式）

登录后台管理 → 基本设置 → 博客主题风格 → 选择主题 → 保存

**即时生效，无需重启！**

## 可用主题

- **Default**：原有默认风格
- **Indigo**（推荐）：参照 yscoder.github.io，Material Design 风格
- **Hexo**：简洁清新，卡片式布局
- **Astro**：现代简约风格
- **Vercel**：Vercel 风格设计

## 技术实现

### 主题选择逻辑

```python
def _get_blog_theme():
    """获取当前博客主题配置 - 优先从用户Cookie读取，其次从数据库，最后从配置文件"""
    # 1. 优先从用户 Cookie 读取
    user_theme = request.cookies.get('user_blog_theme')
    valid_themes = ['default', 'indigo', 'hexo', 'astro', 'vercel']
    if user_theme and user_theme in valid_themes:
        return user_theme
    
    # 2. 从数据库读取系统默认主题
    theme = SystemSetting.get('blog_theme')
    
    # 3. 如果数据库没有，从配置文件读取
    if not theme:
        config = current_app.config.get('APP_CONFIG', {})
        theme = config.get('blog_theme', 'default')
    
    # 验证主题是否有效
    return theme if theme in valid_themes else 'default'

def _get_template_name(base_name):
    """根据主题获取模板名称"""
    theme = _get_blog_theme()
    if theme == 'hexo':
        return f"{base_name}_hexo.html"
    elif theme == 'indigo':
        return f"{base_name}_indigo.html"
    elif theme == 'astro':
        return f"{base_name}_astro.html"
    elif theme == 'vercel':
        return f"{base_name}_vercel.html"
    return f"{base_name}.html"
```

### 数据库存储

主题设置存储在 `system_settings` 表中：

```sql
INSERT INTO system_settings (key, value) VALUES ('blog_theme', 'indigo');
```

## 自定义主题

### 修改 Indigo 主题颜色

编辑 `static/css/blog-indigo-new.css`：

```css
:root {
    --sidebar-bg: #34495e;      /* 侧边栏背景 */
    --link-color: #3f51b5;      /* 链接颜色 */
    --content-bg: #f8f9fa;      /* 内容背景 */
}
```

### 修改 Hexo 主题颜色

编辑 `static/css/blog-hexo.css`：

```css
:root {
    --primary: #49b1f5;      /* 主色调 */
    --accent: #ff7242;       /* 强调色 */
}
```

## 注意事项

1. 主题切换不影响文章内容和数据库数据
2. 修改 CSS 后需要清除浏览器缓存
3. 所有主题都支持响应式设计
4. 推荐使用 Indigo 主题获得最佳体验

## 问题排查

### 主题没有切换？

1. 检查数据库中的 `blog_theme` 设置
2. 清除浏览器缓存（Ctrl+F5）
3. 检查模板文件是否存在

### 样式显示异常？

1. 检查 CSS 文件是否正确加载
2. 查看浏览器控制台是否有错误
3. 确认模板文件路径正确
